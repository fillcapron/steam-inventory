Интернет приложение для проверки инветаря пользователя **Steam**:
-Общее кол-во предметов
-Подробная информация о предмете + цена на площадках
-Сортировка предметов по кол-ву, названию, редкости
-Поиск предметов

**Stack:**
+ TypeScript
+ ReactJS + hooks
+ React Router
+ Redux
+ Redux thunk
+ SCSS
